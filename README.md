# GEX1207 - Estruturas de Dados I

Repositório destinado ao armazenamento das atividades, exercícios e trabalhos desenvolvidos na disciplina **GEX1207 - Estruturas de Dados I**, realizada no semestre **2025/2**, durante o **2º semestre do curso de Ciência da Computação** da **Universidade Federal da Fronteira Sul - UFFS, Campus Chapecó**.

A disciplina foi desenvolvida utilizando a linguagem de programação **C**, com foco no estudo e implementação de estruturas de dados fundamentais.

---

## 📚 Sobre a Disciplina

A disciplina de **Estruturas de Dados I** tem como objetivo apresentar conceitos fundamentais para a organização, manipulação e armazenamento eficiente de dados em programas computacionais.

Durante o semestre, foram estudados conteúdos como:

- Funções em C;
- Ponteiros;
- Vetores;
- Matrizes;
- Alocação dinâmica de memória;
- Structs;
- Tipos Abstratos de Dados;
- Listas encadeadas;
- Listas duplamente encadeadas;
- Filas;
- Pilhas;
- Árvores;
- Heaps;
- Desenvolvimento de trabalhos práticos em C.

## 📌 Conteúdo das Pastas

### `01-FUNÇÕES`

Atividades introdutórias envolvendo a criação e utilização de funções em linguagem C.

### `02-PONTEIROS`

Exercícios relacionados ao uso de ponteiros, endereços de memória e manipulação indireta de variáveis.

### `03-VETORES`

Atividades envolvendo vetores, percorrimento de arrays, leitura, escrita e manipulação de dados sequenciais.

### `04-MATRIZES`

Exercícios com matrizes, acesso por índices, percorrimento bidimensional e manipulação de dados em tabelas.

### `05-CALCULADORA`

Implementação de uma calculadora utilizando conceitos básicos da linguagem C.

### `06-STRUCTS`

Atividades envolvendo o uso de `structs` para agrupar diferentes dados em uma única estrutura.

### `07-TAD`

Estudos e implementações de **Tipos Abstratos de Dados**, separando a interface da implementação.

### `08-ALOCAÇÃO-DINAMICA`

Exercícios utilizando alocação dinâmica de memória com funções como `malloc`, `calloc`, `realloc` e `free`.

### `09-LISTA-ENCADEADA`

Implementação e manipulação de listas simplesmente encadeadas.

### `10-LISTA-DUPLAMENTE-ENCADEADA`

Implementação de listas duplamente encadeadas, permitindo navegação nos dois sentidos da estrutura.

### `11-Filas`

Atividades relacionadas à estrutura de dados fila, utilizando o princípio **FIFO**.

### `12-PILHA`

Implementação da estrutura de dados pilha, baseada no princípio **LIFO**.

### `12-ARVORE`

Exercícios e implementações envolvendo árvores, com foco em organização hierárquica de dados.

### `13-HEAP`

Atividades envolvendo a estrutura de dados heap e suas aplicações.

### `T1-INTRODUÇÃO-C`

Primeiro trabalho da disciplina, com foco na introdução à linguagem C.

### `T2-TAD`

Segundo trabalho da disciplina, envolvendo a aplicação de Tipos Abstratos de Dados.

### `TRABALHO-FINAL`

Trabalho final da disciplina, reunindo conceitos estudados ao longo do semestre.

## ▶️ Como Compilar e Executar

Para compilar um arquivo em C, acesse a pasta desejada e utilize o comando:

```bash
gcc nome_do_arquivo.c -o programa
```

Em seguida, execute o programa com:

```bash
./programa
```

Exemplo:

```bash
cd 09-LISTA-ENCADEADA
gcc main.c -o programa
./programa
```

Caso o projeto possua múltiplos arquivos `.c`, a compilação pode ser feita da seguinte forma:

```bash
gcc main.c arquivo1.c arquivo2.c -o programa
./programa
```
